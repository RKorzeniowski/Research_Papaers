                            Yuri Broze • 6 months ago

                            I'd guess that the increasing correlation between the cryptocurrencies is due to the participation of hedge funds. Pair trading and other statistical arbitrage techniques can extract value from correlated equities (or currencies) by betting that short-term fluctuations in price differences will eventually even out. They're taking advantage of inefficiencies in the market.

                            The more arbitrage taking place (as more arbitrageurs enter the market), the more and more correlated these currencies should become. The more correlated these currencies become, the less and less opportunity for statistical arbitrage there is. The market's become more efficient.

                            See this article on pair-trading cryptocurrencies: https://megatrader.org/en/p...

                            Implementation of strategies on Quantopian: https://www.quantopian.com/...

                                      As per another comment, I've been trying to alter the code so that it should (maybe, hopefully, might) update prices to the most recent (yesterday).

                                      Edit: I'm not sure but I think the below only works if there is already something cached, I'm not sure it would work if there wasn't already something there.

                                      def get_quandl_data(quandl_id):
                                      '''Download and cache Quandl dataseries'''
                                      cache_path = '{}.pkl'.format(quandl_id).replace('/','-')

                                      f = open(cache_path, 'rb')
                                      df = pickle.load(f)
                                      if df.index[-1]!= datetime.now():
                                      df.append(quandl.get(quandl_id, start_date = str(btc_usd_bnc.index[-1]), end_date = str(datetime.now()), returns="pandas"))
                                      print('Loaded {} from cache'.format(quandl_id))
                                      print('Downloading {} from Quandl'.format(quandl_id))
                                      df = quandl.get(quandl_id, returns="pandas")
                                      df.to_pickle(cache_path)
                                      print('Cached {} at {}'.format(quandl_id, cache_path))
                                      return df

                                      btc_usd_bnc = get_quandl_data('BNC3/GWA_BTC')
                                      btc_usd_bnc = btc_usd_bnc.drop_duplicates()

                                      print(btc_usd_bnc.tail(5))

                                      Notes:
                                      1) Instead of using the BTCUSD price from multiple exchanges as Patrick's so elegantly done, I took the lazy way and used the BNC Global weighted average.
                                      2) I've changed some of the names, just in case anyone deems this useful you may have to tinker with it a bit to match up.
                                      3) This seems to work for me but I can't stress how much of a python noob I am so if it's not right, I apologize.
                                      4) I should probably have used a timedelta thing to check if the last date was yesterday, not today as today's close price doesn't exist yet but I've already pasted it now.
