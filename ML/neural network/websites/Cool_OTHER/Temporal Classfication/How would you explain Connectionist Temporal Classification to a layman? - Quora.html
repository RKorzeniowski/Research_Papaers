Connectionist Temporal Classification means what it says on the tin but it is perhaps best explained backwards;)

We all understand the idea of classification. Looking at a supermarket scene we can see a number of objects and can classify them as shelves, people, trolleys etc. That is the classification part.

Temporal classification is when we need a sequence of things in order to be able to classify an object. So if we are given a single note from a symphony we cannot say what Beethoven symphony it comes from. Give use a sequence, perhaps a bar or two then its easy for some.

The final word, connectionist is the name of a set of techniques that can be used for classification. A more common name for it is neural network but connectionist gives the flavour of having a number of units that are connected together and cooperatively perform a task.

Answered Nov 6

Neural Networks like RNN are used to predict sequence data. One example is speech recognition. But, the problem is , that with temporal data (data that varies with time) , the prediction of the output (labels) may need an order or sequence. Which, means if the RNN is predicting in context of Machine learning and it predicts “Neural” as an output, it’s highly likely it will predict a “Network” as the second label. To solve this we have used Hybrid Systems where the outputs that are predicted by RNN are passed through a HMM to predict the labels that makes most sense in sequence. CTC helps to achieve this by making label predictions at any point in the input sequence, so that the overall sequence of label predicion is correct.
